# Alhusaini Lab Neuroimaging

Documentation and pipeline code for neuroimaging preprocessing and analysis projects in the Alhusaini Lab at Brown University.

## Contents

- **oscar_preprocessing.md** — General preprocessing pipeline for all projects (XNAT2BIDS → FreeSurfer)
- **thalamocortical/** — Thalamocortical structural connectivity pipeline (Lausanne + THOMAS → Connectome)

## Getting Started

All pipelines run on the OSCAR HPC cluster at Brown CCV. Start with `oscar_preprocessing.md` before any project-specific pipeline.
